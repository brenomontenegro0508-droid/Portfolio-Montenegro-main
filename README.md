# 🎨 Portfólio - Breno Montenegro

Bem-vindo ao meu portfólio de Front End! Este projeto é uma apresentação profissional de minhas habilidades, competências e objetivos como desenvolvedor.

## 📌 Sobre o Projeto

Um portfólio moderno e responsivo desenvolvido com **HTML5**, **CSS3** e **JavaScript vanilla**. Apresenta:

- 🎯 Uma página inicial com informações sobre meu perfil
- 📖 Página sobre com competências técnicas e interesses
- 🚀 Página de objetivos profissionais
- 🌙 Tema claro e escuro com persistência em localStorage
- 📱 Design totalmente responsivo (mobile first)
- ✉️ Formulário de contato integrado com FormSubmit

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Descrição |
|-----------|-----------|
| **HTML5** | Estrutura semântica e bem organizada |
| **CSS3** | Layout moderno com Flexbox/Grid, animações e responsividade |
| **JavaScript** | Interatividade, tema dark/light e manipulação do DOM |
| **FormSubmit** | Integração para envio de formulários |
| **Google Fonts** | Tipografia moderna (Poppins) |

## 🚀 Como Usar

### Visualizar localmente

1. Clone o repositório:
```bash
git clone https://github.com/brenomontenegro0508-droid/Portfolio-Montenegro-main.git
```

2. Navegue até o diretório:
```bash
cd Portfolio-Montenegro-main
```

3. Abra `index.html` em seu navegador (ou use um servidor local):
```bash
# Com Python 3
python -m http.server 8000

# Ou com Node.js (http-server)
npx http-server
```

4. Acesse em `http://localhost:8000`

### Personalizar o Portfólio

- **Alterar dados pessoais:** Edite os arquivos `.html` com suas informações
- **Mudar cores:** Modifique as variáveis CSS em `style.css` (`:root`)
- **Adicionar foto:** Coloque sua imagem em `img/profile.png`
- **Atualizar email de contato:** Altere em `index.html` linhas 136, 146 e 151
